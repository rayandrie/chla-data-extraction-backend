About the Project

The Childrenâ€™s Hospital of LA has developed a tool to help clinical researchers mass pull data from public databases like the American Community Survey (ACS). Previously, researchers had to tediously and manually obtain data for each element of their dataset one by one, but now with this tool they can send in their input dataset as a csv and the tool will append information to that input pertaining to a user-selected database parameter.

This tool was built to allow researchers to batch queries on public databases and online calculators. It was developed by Sarah Lu, Raymond Andriejanssen, and Sophia Hu as part of the CS 401 class at USC. Their project mentor is Juan Espinoza, MD. You can contact the project team here.
